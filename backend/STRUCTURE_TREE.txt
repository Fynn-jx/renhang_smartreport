🎯 开发流程指南：
────────────
1️⃣ 新增数据库模型 → models/
2️⃣ 定义 Schema → schemas/
3️⃣ 实现业务逻辑 → services/
4️⃣ 创建 API 端点 → api/endpoints/
5️⃣ 注册路由 → api/v1/api.py
6️⃣ 编写测试 → tests/


📌 重要规则：
────────────
✅ API 层 → 服务层 → 模型层
❌ 不跨层调用
❌ 不在 API 层写业务逻辑
❌ 不在服务层返回 HTTP 响应


🚀 下一步：
────────────
1. 配置 .env 文件
2. 安装依赖: pip install -r requirements.txt
3. 启动数据库: PostgreSQL + Redis
4. 初始化数据库: alembic upgrade head
5. 启动服务: python main.py
6. 访问文档: http://localhost:8000/docs
